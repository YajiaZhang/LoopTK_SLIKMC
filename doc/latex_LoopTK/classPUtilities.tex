\subsection{Detailed Description}
Contains miscellaneous utility methods for string manipulation, text file I/O, system interaction, and so on. These methods are made public in case they are valuable to users of {\tt \doxyref{Loop\-TK}{p.}{classLoopTK}} but are generally intended for internal use.
