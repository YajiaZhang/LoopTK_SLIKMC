\subsection{Detailed Description}
A subclass of \doxyref{PChain}{p.}{classPChain} which makes strong assumptions about the properties the loop should have. Specifically, convenience methods are provided to selectively manipulate the backbone and sidechain or to retrieve anchor/end effector atoms at the amino or carboxy terminus. This class should be used when using the builtin resources with Loop\-TK.

Usage of a \doxyref{PProtein}{p.}{classPProtein} must follow the following sequence of steps:

\begin{enumerate}
\item {\bf Construction} - all residues are added to chain. Use the \doxyref{PProtein::Add\-Residue()}{p.}{classPChain_8a42f1e7a59dbe7d3e9e463906054ef2} methods to do this.\item {\bf Finalization} - finishes construction of the protein and prepares it for manipulation. Use \doxyref{PProtein::finalize()}{p.}{classPChain_4cb2a0f087cd2c70dca1349b15c12d53} to do this.\item {\bf Analysis} - the full range of functionality of \doxyref{PProtein}{p.}{classPProtein} can be used at this point. However, no more residues can be added.\end{enumerate}


Creating a \doxyref{PProtein}{p.}{classPProtein} using \doxyref{PDBIO}{p.}{classPDBIO} automatically completes the first two steps.
