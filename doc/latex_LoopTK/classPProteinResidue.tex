\subsection{Detailed Description}
This class represents a residue within \doxyref{PProtein}{p.}{classPProtein}, so it makes assumptions on the contents within. More specifically, this residue has a backbone and sidechain and depends on the configuration data provided with \doxyref{Loop\-TK}{p.}{classLoopTK}. Do not create \doxyref{PProtein\-Residue}{p.}{classPProteinResidue} directly - use \doxyref{PProtein::Add\-Residue}{p.}{classPChain_8a42f1e7a59dbe7d3e9e463906054ef2} instead. See \doxyref{PResidue}{p.}{classPResidue}, the superclass, for more operations.
