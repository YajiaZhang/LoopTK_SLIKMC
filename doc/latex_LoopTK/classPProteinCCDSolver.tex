\subsection{Detailed Description}
Simplifies the construction of a CCD solver for a {\tt \doxyref{PProtein}{p.}{classPProtein}}, allowing the user to pass only a pointer to the loop and, if desired, the direction as a {\tt Protein\-Side} parameter. The standard {\tt \doxyref{PCCDSolver}{p.}{classPCCDSolver}} functions can then be used to perform cyclic coordinate descent.
